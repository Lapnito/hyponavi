## 계산

`M = P × [r(1+r)^n / ((1+r)^n − 1)]` — 은행 상환 스케줄과 원 단위로 일치.

## 상환할까 투자할까?

세후 수익 > 모기지 금리 → 투자. 낮음 → 상환. 템플릿이 차이를 보여줌.

## 스트레스 테스트

5년 고정 4.5% — 6년차 6.5%면 월 상환은 "단지" 2pt 아니라 잔액에 따라 25–35% 상승.
